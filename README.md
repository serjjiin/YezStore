# Yez Store

Loja virtual colaborativa de artesanato. Múltiplos artesãos, vitrine única, operação centralizada.

**Documentação completa:** [`PROJETO.md`](./PROJETO.md)

---

## Setup local

```bash
# 1. Instalar dependências
npm install

# 2. Configurar variáveis de ambiente
cp .env.example .env.local
# Preencher os valores em .env.local

# 3. Rodar em desenvolvimento
npm run dev       # http://localhost:3000
```

## Comandos

```bash
npm run dev        # servidor de desenvolvimento
npm run build      # build de produção
npm run lint       # ESLint
npm test           # testes (Vitest)
```

## Stack

Next.js 16 · React 19 · Supabase · Mercado Pago · Melhor Envio · Zustand · Vitest
